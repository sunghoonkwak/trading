# trading_state.md

이 모듈은 애플리케이션의 공유 상태 저장소를 제공합니다.

## Purpose (목적)
웹소켓 콜백과 메인 UI 루프 등 서로 다른 컴포넌트 간에 공유되어야 하는 전역 런타임 상태를 유지하는 것입니다.

## Function (기능)

### stock_data_state
웹소켓으로부터 수신된 최신 실시간 데이터를 저장하는 데 사용되는 딕셔너리입니다.
- 티커 또는 시장별 TR 코드를 키로 사용합니다.
- 전체 업데이트가 발생하지 않은 시점에도 UI에서 가장 최근의 가격이나 주문 상태를 표시할 수 있도록 지원합니다.
#### input
- `None` (변수 직접 접근).
#### output
- `dict`: 현재 전역 상태 정보.
