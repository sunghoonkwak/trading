# trading_state.md

이 모듈은 애플리케이션의 공유 상태 저장소를 제공합니다.

## Purpose (목적)
웹소켓 콜백과 메인 UI 루프 등 서로 다른 컴포넌트 간에 공유되어야 하는 전역 런타임 상태를 유지하는 것입니다.

## Function (기능)

### load_stock_data / save_stock_data
`stock_data.json` 파일을 통해 실시간 시세 데이터를 영속화(Persistence)합니다. 프로그램 재시작 시 시세 정보를 빠르게 복구하여 공백 없는 모니터링을 지원합니다.
- **데이터 신선도 체크**: 로딩 시 파일에 기록된 시간과 현재 시간을 비교하며, **5분 이상 경과한 노후 데이터는 자동으로 무시**하여 데이터의 정합성을 보장합니다.

### notify_data_received
최초 실시간 데이터 수신 시 호출되어 60초 주기의 **백그라운드 자동 저장 스레드**를 활성화합니다.

### stop_periodic_save
프로그램 종료 시 호출되어 자동 저장 스레드를 안전하게 중지하고, 현재 상태의 최종본을 파일에 저장합니다.
