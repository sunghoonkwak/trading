# Value Averaging (VA) 텔레그램 명령어

이 모듈은 Value Averaging 전략에 기반한 주문을 텔레그램에서 대화형으로 실행할 수 있게 합니다.

## 명령어

### `/value_averaging`
- **설명**: 현재 설정된 모든 VA 전략에 대해 오늘의 예상 주문을 확인하고, 일괄 실행합니다.
- **작동 방식**:
    1. 모든 전략의 리포트를 생성하여 한 번에 보여줍니다.
    2. 주문이 필요한 항목(Pending)이 있으면 `🚀 Execute All` 버튼을 표시합니다.
    3. `🚀 Execute All` 버튼을 클릭하면 대기 중인 모든 주문을 순차적으로 실행하고 결과를 리포트로 보여줍니다.
    4. `❌ Cancel` 버튼을 누르면 작업을 취소합니다.
- **특이 사항**:
    - 휴장일인 경우 `🚫 Market Holiday` 안내 메시지만 보여줍니다.
    - 이미 주문이 실행된 항목은 리포트 요약에 `✅ Executed`로 표시됩니다.
    - 주문이 필요 없는 항목(Hold)은 `⏸️ Hold`로 표시됩니다.
    - 세션은 60초 동안 응답이 없으면 만료됩니다.
