# Value Averaging (VA) 텔레그램 명령어

이 모듈은 Value Averaging 전략에 기반한 주문을 텔레그램에서 대화형으로 실행할 수 있게 합니다.

## 명령어

### `/value_average`
- **설명**: 현재 설정된 모든 VA 전략에 대해 오늘의 예상 주문을 확인하고, 사용자 승인 하에 주문을 실행합니다.
- **작동 방식**:
    1. 모든 전략의 리포트를 생성합니다.
    2. 주문이 필요한 항목(Pending)이 있으면 하나씩 순차적으로 보여줍니다.
    3. 사용자가 `✅ Yes`를 누르면 주문이 실행되고 결과가 기록됩니다.
    4. `❌ No`를 누르면 해당 항목은 건너뛰고 기록에 'Skipped'로 남습니다.
    5. 모든 항목에 대한 처리가 끝나면 최종 요약 리포트를 보여줍니다.
- **특이 사항**:
    - 휴장일인 경우 주문 확인 과정을 생략하고 휴장 안내 리포트만 보여줍니다.
    - 이미 주문이 실행된 항목은 리포트 요약에 `✅ Executed`로 표시됩니다.
    - 주문이 필요 없는 항목(Hold)은 `⏸️ Hold`로 표시됩니다.
    - 세션은 60초 동안 응답이 없으면 만료됩니다.
